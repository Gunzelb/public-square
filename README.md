## About the Project

>A social network with public and private feeds for sharing your thoughts and ideas with the world and your closest friends

### Built With
- MERN Stack 
  - MongoDB/Mongoose 
  - Express 
  - React 
  - Node
- Chakra UI  
- GraphQL 
- Apollo 
- Bcrypt 
- JSON Web Token (JWT)
- Heroku
